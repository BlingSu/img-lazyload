# 一个轻量图片懒加载插件

img-lazyload是一个简单的，轻量的图片懒加载脚本

## 正文

### Html
```html
<img class="lazyload" data-src="test.jpg">
```

### Javascript
```js
<script type="text/javascript" src="./index.js"></script>
<script type="text/javascript">
  var lazyLoad = new LazyLoad('.lazyload', {})
</script>
```
